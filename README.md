# Desktop Pets

Initial Commit

## Installation

Requirements: Git, Pip, Python

- Clone the repository
```
git clone https://github.com/Puce807/desktop-pets
```

- Install package requirements
```
pip install -r requirements.txt
```

- Run project
```
python main.py
```
